变量


1. 变量声明: 

var varName;

2. 变量赋值: 

varName = 2;

3. 基本数据类型:

  • 数字类型(Number)--变量存储数字: 
        var num = 2;
  • 字符串类型(String)--变量存储字符串: 
        var str = 'Hello';
  • 布尔类型(Boolean)--变量存储布尔值: 
        var right = true
  • 数组(Array)--创建数组： 
        var nums;
    nums = [2, 6, 8];
  • ...
